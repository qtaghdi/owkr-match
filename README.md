# OWKR Match

OWKR 내전 밸런스 조정 웹사이트입니다.

## 어떻게 사용하나요?

### 1단계: 디스코드에서 티어 정보 복사하기

디스코드에서 참가자들이 올린 티어 정보를 드래그해서 복사합니다 (Ctrl+C 또는 Cmd+C).

```
테스터1#11853 다5/다1/다5
테스터2#38848 다3/마4/다4
테스터3#34981 미배치(골)/미배치(플)/플2
테스터4#31207 그5!/마1!/마4
```

> **티어 약어 설명**
> - `브` = 브론즈, `실` = 실버, `골` = 골드
> - `플` = 플래티넘, `다` = 다이아몬드, `마` = 마스터
> - `그` = 그랜드마스터, `챔` = 챔피언
> - 숫자는 등급 (1~5)
> - 순서는 항상 **탱커/딜러/힐러**

### 2단계: 웹사이트에 붙여넣기

1. [OWKR Match](https://owkr-match.vercel.app/) 웹사이트에 접속합니다
2. **디스코드 파싱** 탭의 텍스트 박스에 붙여넣기 합니다 (Ctrl+V 또는 Cmd+V)
3. **플레이어 추가하기** 버튼을 클릭합니다

### 3단계: 팀 생성하기

1. 대기 명단에 **10명**이 모이면 **팀 짜기** 버튼이 활성화됩니다
2. 버튼을 클릭하면 자동으로 밸런스 맞춘 두 팀이 생성됩니다

### 4단계: 필요시 선수 교체하기

결과가 마음에 안 들면 선수를 직접 바꿀 수 있습니다.
- 교체하고 싶은 선수 A를 클릭
- 교체할 선수 B를 클릭
- 두 선수의 위치가 바뀝니다

### 5단계: 결과 공유하기

**이미지로 복사** 버튼을 클릭하면 결과가 이미지로 복사됩니다. 디스코드에 바로 붙여넣기 하면 됩니다.

---

## 선호 포지션 설정하기

특정 포지션만 하고 싶은 사람이 있다면, 해당 티어 뒤에 **느낌표(!)**를 붙이면 됩니다.

```
재봉이#31207 그5!/마1!/마4
```

위 예시에서 `그5!`는 "탱커 그랜드마스터 5등급, **탱커 선호**"라는 뜻입니다.
시스템이 최대한 해당 포지션에 배치하려고 노력합니다.

## 자주 묻는 질문

**Q: 10명이 안 모이면 어떻게 하나요?**
A: 10명이 모여야 팀을 나눌 수 있습니다. 대기 명단에서 몇 명이 더 필요한지 확인하세요.

**Q: 데이터가 저장되나요?**
A: 플레이어 정보는 브라우저에 24시간 동안만 저장됩니다. 다음 날이 되면 자동으로 삭제됩니다.

**Q: 미배치는 어떻게 입력하나요?**
A: `미배치(골)` 형식으로 입력하면 됩니다. 괄호 안은 예상 티어입니다.

**Q: 수동으로 한 명씩 추가할 수도 있나요?**
A: 네, **수동 입력** 탭에서 직접 티어를 선택해서 추가할 수 있습니다.

**Q: 디스코드를 통한 로그인이 안됩니다.**
A: 이 서비스는 OWKR 관리자만 등록이 가능합니다. 관리자지만 로그인이 안될경우, discord: qtaghdi에 문의 부탁드립니다.
