# OWKR Match (오버워치 2 팀 밸런싱 도구)

본 프로젝트는 오버워치 2 내전(스크림) 진행 시, 참여 인원의 경쟁전 등급을 기반으로 공정한 팀 분배를 수행하기 위해 제작된 웹 애플리케이션입니다.

디스코드 채팅 로그를 통한 대량 데이터 입력, 특정 포지션 고정, 경쟁전 점수 기반의 자동 밸런싱 알고리즘을 지원합니다.

## 주요 기능

1.  **텍스트 파싱 및 자동 입력**
    * 정형화되지 않은 텍스트(예: 디스코드 채팅 로그)에서 배틀태그, 역할군, 티어 정보를 자동으로 추출하여 대기 명단에 등록합니다.
    * 날짜, 시간, 기타 불필요한 대화 내용은 자동으로 무시됩니다.

2.  **포지션 고정 (Role Lock)**
    * 특정 플레이어를 원하는 역할군에 강제로 배정할 수 있습니다.
    * 밸런싱 알고리즘은 고정된 포지션을 최우선으로 고려하여 팀을 구성합니다.

3.  **밸런싱 알고리즘**
    * 총 10명의 플레이어를 5명씩 두 팀으로 나누는 모든 경우의 수와 각 팀 내부의 역할 배정 순열을 계산합니다.
    * 양 팀의 점수 격차가 최소화되는 최적의 조합을 도출합니다.

4.  **팀원 수동 교체**
    * 알고리즘 결과가 생성된 후, 관리자가 직접 플레이어를 클릭하여 서로의 위치를 맞바꿀 수 있습니다.

## 사용 가이드

### 1. 플레이어 데이터 입력
웹사이트 하단의 '디스코드 채팅 붙여넣기' 영역을 통해 데이터를 입력하거나, 상단의 입력 폼을 통해 수동으로 추가할 수 있습니다.

**지원하는 텍스트 형식:**
* **기본 형식:** `닉네임#태그 포지션 등급`
* **약어 지원:** `탱/딜/힐`, `다3/플1/골2` 등의 약어를 인식합니다.
* **순차 입력:** 포지션을 명시하지 않고 등급만 나열할 경우, 자동으로 `탱커 → 딜러 → 힐러` 순서로 인식합니다.

**입력 예시:**
```text
PlayerOne#1234 탱커 마스터3 / 딜러 다이아1 / 힐러 마스터5
PlayerTwo#5678 다3 플2 골1
```

### 2. 포지션 고정 (선호 포지션 지정)
특정 플레이어를 반드시 해당 포지션에 배정하고 싶을 경우, 등급 뒤에 느낌표(`!`)를 입력합니다.

* `UserA#1234 딜러 다이아1!` → 해당 유저는 **반드시 딜러**로 배정됩니다.
* `UserB#5678 힐러 마스터3!` → 해당 유저는 **반드시 힐러**로 배정됩니다.

밸런싱 알고리즘은 고정된 포지션을 최우선 순위로 고려하여 팀을 구성합니다.

### 3. 팀 생성 및 조정
1.  대기 명단에 총 10명의 플레이어가 등록되어야 합니다.
2.  `팀 짜기` 버튼을 클릭하면 계산이 수행됩니다.
3.  결과 화면에서 `Team 1`의 선수와 `Team 2`의 선수를 각각 클릭하면 두 선수의 소속 팀이 교체됩니다.
