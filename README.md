# OWKR Match — 사용 방법

> 오버워치2 내전 팀 매칭 도구  
> 입력된 10명의 티어를 기반으로 **탱 > 딜 > 힐** 우선도로 전수 탐색하여 두 팀을 자동으로 구성합니다.  
> (가중치/시도횟수 없음 · 정확히 10명일 때만 팀 생성)

---

## 1) 기본 흐름

1. **플레이어 추가**
   - 폼으로 닉네임/티어를 선택하고 `추가` 버튼을 누릅니다.
   - 또는 아래 “붙여넣기 추가”에 여러 줄을 붙여넣고 `파싱해서 추가` 버튼을 누릅니다.

2. **팀짜기**
   - `팀짜기` 버튼을 누르면 10명을 기준으로 팀을 자동 구성합니다.
   - 팀 구성은 **탱 > 딜 > 힐** 우선도를 최대화하도록 전수 탐색합니다.
   - 정확히 **10명**이 아니면 팀을 만들 수 없습니다.

3. **편집(선택 사항)**
   - 표에서 드래그앤드롭 혹은 클릭-스왑으로 팀/포지션을 바꿀 수 있습니다.
   - `자동 균형 재계산` 버튼으로 현재 10명을 기준으로 최적 분할을 다시 계산할 수 있습니다.
   - `선택 해제`로 클릭-스왑의 첫 선택을 초기화합니다.

---

## 2) 입력 형식 (붙여넣기)

- 한 줄에 한 명씩
- 형태 A: `닉#태그 /다5/플3/실2` (역할 라벨 없이 **딜/탱/힐 순서**로 자동 해석)
- 형태 B: `닉#태그 / 탱 다5 / 딜 골2 / 힐 플4` (역할 라벨 포함)
- 역할 라벨 허용: `탱|탱커|tank|t`, `딜|딜러|dps|d`, `힐|힐러|sup|support|s`
- 티어 토큰(앞부분만 적어도 인식):  
  - 브(브론즈), 실(실버), 골(골드), 플/플레(플래티넘), 다(다이아), 마(마스터), 그/그마(그랜드마스터), 챔(챔피언)  
  - 디비전은 1~5 (예: `다이아 3`, `플2`, `마 5`)

**예시**

PlayerOne#1111 /다5/다4/다4
PlayerTwo#2222 / 탱 다5 / 딜 골2 / 힐 플4



> 붙여넣기 파서는 불필요한 공백/풀와이드 슬래시(`／`)도 처리하며,  
> 역할 라벨이 전혀 없으면 **딜 → 탱 → 힐** 순서로 자동 매핑합니다.

---

## 3) 편집 모드 (업데이트된 수정 로직)

### 드래그앤드롭
- 슬롯(한 행의 팀/포지션 칸)을 **끌어서** 다른 슬롯에 **떨어뜨리면** 두 플레이어가 스왑됩니다.

### 클릭-스왑
- 첫 번째 슬롯을 **클릭**해 선택(하이라이트) → 두 번째 슬롯을 클릭하면 **스왑**됩니다.
- `선택 해제` 버튼으로 첫 선택을 초기화할 수 있습니다.

### **가로형 타 티어 표시 (NEW)**
- 편집 모드일 때, 각 슬롯에 **닉네임 오른쪽**에 **모든 포지션 티어**를 **가로로** 보여줍니다.
- 표기 형태(짧은 표기): `T:다3 / D:플2 / S:골1`  
  - `다3` = 다이아 3, `플2` = 플래티넘 2 처럼 축약 표기  
- 이 방식으로 행 높이를 늘리지 않고 한 줄에서 모두 비교할 수 있어 **가독성과 편집성이 향상**되었습니다.

> 이전 버전처럼 현재 포지션만 굵게/별도 라인으로 표기하면 행이 높아져 UI가 깨질 수 있어,  
> **한 줄 가로 표시**로 통일했습니다.

---

## 4) 우선도/점수화 로직

- 내부 점수화: 티어(브~챔)와 디비전(1~5)을 숫자로 환산
- 팀 내부 배치 탐색 시 **탱 합계 > 딜 합계 > 힐 합계** 순으로 렉시코그래픽 비교
- 두 팀 분할은 10명 중 5명 조합을 전수 탐색하여, 양 팀의 우선도 값 차이가 최소가 되도록 선택

---

## 5) 버튼 설명

- **팀짜기**: 현재 목록(정확히 10명)으로 자동 팀 구성
- **편집 모드**: 편집 켜기/끄기 토글(드래그/클릭-스왑 사용 가능)
- **자동 균형 재계산**: 현재 두 팀의 10명을 재수집하여 최적 재분할
- **선택 해제**: 클릭-스왑의 첫 선택을 초기화

---

## 6) 자주 묻는 질문

- **Q. 10명 미만/초과인데 팀이 안 만들어져요.**  
  A. 설계상 **정확히 10명**일 때만 팀을 생성합니다.

- **Q. 붙여넣기가 실패해요.**  
  A. 각 줄에 `닉#태그`와 세 포지션의 티어가 모두 있어야 합니다.  
     라벨이 없다면 순서가 `딜/탱/힐`인지 확인하세요.  
     티어는 반드시 디비전(1~5)을 포함해야 합니다. (예: `다3`, `플2`, `골1`)

- **Q. 현재 포지션만 강조하고 싶어요.**  
  A. 가로형 표시(한 줄)에 강조 스타일을 추가할 수 있습니다. 필요하면 알려주세요.
